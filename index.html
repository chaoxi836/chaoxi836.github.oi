GitHub Pages 部署指南

1

创建GitHub仓库

登录GitHub账号，创建一个新仓库

仓库名必须为：你的用户名.github.io

设置为公开仓库，并初始化README文件

2

上传网站文件

将您的网站文件上传到仓库中

确保包含一个index.html文件作为首页

git add .
git commit -m "初始化网站"
git push origin main
3

启用GitHub Pages

进入仓库设置 > Pages

选择分支（通常为main或master）

保存设置，等待部署完成

访问：https://你的用户名.github.io
